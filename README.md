# Sistema de Administracion de Usuarios
Desarrollo de un sistema escalable para la administración y gestión de usuarios con diferentes roles.

## Tecnologías y Conceptos Usados
- JAVA
- Base de Datos Relacionales MySQL
- JPA
- Swing
- Maven
